Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main
import (
	"fmt"
)
func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}
func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
Вывод:
[3 2 3]

Cлайс состоит из ссылки на базовый массив с элементами,
из переменной длины слайса, то есть количества элементов, которое в нём содержится,
а также из емкости слайса, то есть количества элементов, которые мы можем записать 
в слайс без его дальнейшего расширения.
При передаче в функцию слайса также передаются ссылки на значения элементов массива, 
на который ссылается данный слайс. 
Таким образом, меняя элементы в функции в копии слайса, 
мы также меняем значения элементов оригинального слайса. 
Но, передавая слайс в функцию не по ссылке, мы не можем изменить длину или емкость слайса.

В данном примере в функции modifySlice сначала мы меняем первый элемент оригинального слайса.
Затем в функции modifySlice с помощью append создается новый слайс i, который перекрывает
внешнюю переменную с тем же названием из функции main. И в дальнейшем все изменения происходят 
с новой переменной i.
```
